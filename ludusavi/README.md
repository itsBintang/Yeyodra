# Ludusavi Integration

This folder contains Ludusavi binary and configuration for game save backup/restore functionality.

## Files
- `ludusavi.exe` - Ludusavi CLI binary (Windows)
- `config.yaml` - Configuration file

## Usage
Ludusavi is used internally by the application to:
- Scan and backup game saves
- Restore game saves from backups
- Support both automatic and manual save locations
